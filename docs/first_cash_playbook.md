# BookRoute First Cash Playbook

## 1. 今天卖什么

不要说：

```text
AI Agent 网盘自动化
```

对外说：

```text
你给我书名/ISBN/阅读清单，我帮你整理合法获取路线：正版、二手、图书馆、公版、开放获取。
```

## 2. 谁会付钱

| 客户 | 痛点 | 你卖的结果 |
| --- | --- | --- |
| 学生 | 老师给书单，不知道怎么买便宜 | 低成本合法购买/借阅路线 |
| 研究生 | 论文/专著到处找不到 | OA、图书馆、馆际互借、出版社路线 |
| 读书会 | 每期书单要统一采购 | 版本确认 + 采购路线 |
| 老师/助教 | 给学生发参考书单 | 合法资源清单 |
| 个人知识库用户 | 想整理资料来源 | 书目元数据 + 获取路径 |

## 3. 第一单价格

| 服务 | 价格 |
| --- | ---: |
| 单本书路线 | 9.9-29 |
| 5 本以内 | 49 |
| 10 本书单 | 99-299 |
| 课程/专题书单 | 299-999 |

## 4. 话术

```text
你把书名/作者/ISBN 发我，我帮你查合法获取路线。
包括正版新书、二手、图书馆、公版书、开放获取版本。
不找盗版网盘，不做代下。
```

## 6. 为什么能赚钱

客户不是为“链接”付钱。

客户为这三件事付钱：

```text
省时间
少踩坑
不碰侵权风险
```
